Detailed Description

Wrapper on top of BME280 driver from Bosh GitHub project.

This file is the wrapper to expose a simple API for getting the temperature, pressure and humidity out of device for a "weather application" configuration. It makes the glue between the Bosh driver and Wirepas SPI driver and staticaly configure the sensor in a given mode.
